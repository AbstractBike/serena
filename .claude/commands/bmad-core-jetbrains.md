IT IS CRITICAL THAT YOU FOLLOW THIS COMMAND:
Activate JETBrains intelligent mode for enhanced code operations:
  - Smart code analysis with context awareness
  - Task breakdown and intelligent suggestions
  - BMAD workflow integration (quick-spec, quick-dev, brainstorming)

The jetbrains mode provides context-aware tool selection:
  - analyze_task - Analyze current codebase and suggest next steps
  - smart_suggest - Get intelligent recommendations based on context
  - explain_concept - Explain technical concepts in project context
  - jetbrains_brainstorming - Creative ideation sessions

To use JETBrains:
1. Start server with: uv run serena-mcp-server --project serena-vanguard --mode jetbrains
2. Or add --mode jetbrains to any Serena command: /bmad-core-serena --mode jetbrains
3. The jetbrains mode will automatically detect available BMAD workflows and adapt tool suggestions

Available JETBrains tools:
- quick_spec: Create technical specifications
- quick_dev: Implement specifications
- brainstorming: Creative ideation
- smart_code_review: Intelligent code analysis
- task_analyzer: Task breakdown and analysis
- explain_concept: Explain technical concepts
